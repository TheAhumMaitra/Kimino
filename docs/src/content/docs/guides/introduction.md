Hello, Welcome to Kimino! I made Kimino for beginners and creative thinkers. Let's say, you want to create a web app (like: Youtube) to launch from your App viewer in Linux. Maybe you just want to create a desktop app entry for a new app, you just downloaded. Don't worry, if you want to create multiple desktop app entry. Kimino is specifically made for these situations. With Kimino you can create your customized app entries.
