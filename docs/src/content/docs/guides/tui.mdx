---
title: Create TUI app shortcut
description: How to create
---

You can create any TUI'S shortcut using `TUI` entry.

### Follow these steps to create a TUI app entry

1. Open Kimino App
2. Go to selection entry and choice TUI
3. Now fill the following fields (such as -`App Name, Tui command, Terminal launcher, icon path`)
3. Now press the `Create Launcher` button and done!
