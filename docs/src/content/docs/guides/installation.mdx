title: How to install Kimino
description: Installation guide for installing Kimino to your Linux system
---

You can install Kimino on almost every Linux distribution.

### Here are some installation options

**Note : All installtion method requires between Python `3.10` , `3.14`**

---

#### Installation using `uv`
---
_Prequirements : Python installed, `uv` installed_

import { Code } from '@astrojs/starlight/components';

##### Using Kimino package

<Code code="uv tool install -U Kimino" lang="bash"/>

##### Using Kimino Github

<Code code="uv tool install -U git+https://github.com/TheAhumMaitra/Kimino.git" lang="bash"/>

#### Installation using `pipx`

<Code code="pipx install Kimino" lang="bash" />
